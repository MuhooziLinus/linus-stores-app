🚀 QUICK START
═══════════════════════════════════════════════════════════════════════════

Option 1: AUTOMATED (Recommended)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  Windows:     setup.bat
  macOS/Linux: bash setup.sh

Option 2: MANUAL SETUP
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  1. cd backend && python -m venv venv && venv\Scripts\activate
  2. pip install -r requirements.txt
  3. Create .env with MySQL credentials
  4. mysql -u root -p < ../database/schema.sql
  5. python run.py
  6. cd frontend && python app.py
  7. Open http://localhost:5001
